# ts-react-redux-webpack-starter
> Starter project for technology stacks
>  + **Framwork:** React-Redux
>  + **Language:** Typescript
>  + **Design:** Material-ui
>  + **Style Management:** SCSS
>  + **Deploy tool:** Webpack

## Usage
1. install dependencies
```
npm install
```
2. run project
```
webpack-dev-server
```
